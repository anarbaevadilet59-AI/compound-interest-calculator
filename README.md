# Compound Interest Calculator
This project is a Python application that demonstrates how compound interest works with monthly contributions.

## What this project does
- Calculates compound interest monthly
- Allows monthly contributions
- Show year-by-year results
- Displays a table and a chart

## Why this project exists
This project was built to deeply understand how compound interest works, including how interest grows on interest over time.
